Receba uma imagem de interface ou layout (por exemplo, uma captura de tela de um site, app ou design do Figma) e adicione bordas finas ao redor de todos os elementos visuais presentes, como textos, imagens, botões, seções e divisores.

Use cores contrastantes para as bordas (como azul, vermelho ou verde) para que cada elemento fique bem demarcado.

O objetivo é evidenciar a estrutura visual da imagem original, revelando um sistema de grade com base nas posições e tamanhos dos elementos.

Importante: Não altere o conteúdo da imagem original — apenas sobreponha as bordas nos elementos detectados. O foco é tornar clara a organização espacial e o layout da imagem.

PROMPT - GERAR GRADE HTML

Input: Envie uma imagem ou diagrama visual em SVG que mostra a estrutura da página, com a grade (grid) destacada de forma clara, como em um wireframe. A grade pode incluir colunas, linhas e espaços entre os elementos. Idealmente, as seções devem estar bem definidas e o layout deve ser simples, sem muitos detalhes adicionais.

Instruções: A IA deve analisar a imagem e gerar um código HTML e CSS que reproduza a estrutura visual da grade apresentada no diagrama. O código gerado deve usar um sistema de grid flexível (por exemplo, CSS Grid ou Flexbox) para criar a disposição correta dos elementos. As instruções específicas incluem:

Identificação das colunas e linhas: A IA deve identificar quantas colunas e linhas existem no diagrama e criar uma estrutura de grid com o número adequado de colunas e linhas.

Tamanhos de colunas e linhas: A IA deve calcular os tamanhos relativos de cada coluna e linha com base no diagrama. Pode usar unidades como fr (fração), px, ou %, conforme o necessário.

Espaçamentos (margens e padding): A IA deve identificar quaisquer espaços entre os elementos (margens e padding) e aplicá-los corretamente no CSS.

Responsividade: A IA deve garantir que o layout seja responsivo, ajustando o número de colunas ou a disposição dos itens conforme o tamanho da tela. Utilize media queries para diferentes tamanhos de dispositivos.

Output: O código HTML e CSS que deve representar a estrutura da grade no diagrama visual enviado.
